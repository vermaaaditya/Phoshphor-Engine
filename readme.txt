=======================
ASCII ART CONVERTER
=======================

A simple, client-side web application that converts any image into beautiful ASCII art. This tool runs entirely in your browser using JavaScript, ensuring that your images remain private and the conversion is instantaneous.


FEATURES
----------------
* Real-time Image Conversion: Upload an image (.jpg, .png, etc.) and see the ASCII art generated instantly.
* Purely Client-Side: All processing happens in your browser. No images are ever uploaded to a server.
* Customizable Output:
    - Resolution Slider: Easily control the level of detail in the final output.
    - Custom Character Ramp: Define your own set of ASCII characters for unique artistic styles.
* Clean & Modern UI: A simple, responsive interface built with Tailwind CSS.


HOW TO USE
----------------
No installation is required.

1.  Download the `index.html` file.
2.  Open the `index.html` file directly in your web browser.
3.  Click "Choose File" to select an image and use the sliders to adjust the output.


TECHNOLOGY STACK
----------------
* HTML5
* Tailwind CSS (via CDN)
* JavaScript (ES6+)
